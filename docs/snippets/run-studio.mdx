Launch the development server with:

<CodeGroup>
```sh npm
npx @actor-core/cli@latest dev
```

```sh pnpm
pnpm exec @actor-core/cli@latest dev
```

```sh yarn
yarn @actor-core/cli@latest dev
```

```sh bun
bunx @actor-core/cli@latest dev
```
</CodeGroup>

This will automatically start your app and open the studio in your browser.

![ActorCore Studio](/images/screenshots/studio/simple.png)

The studio includes **hot-reloading**, **state inspection**, **visual RPC testing**, and more debugging tools.
